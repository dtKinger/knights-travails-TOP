Knight's Travails

This app runs in the developer console.

1. To open the dev console:

Mac users: command + option + J
PC users: control + alt + J

2. Copy + Paste the functions from the API Reference and replace the parameters with values between 0 and 7.

API Reference

Manipulate the position and test the moves of WK1 (White Knight 1) on the chessboard.

getStartingPoint('WK1') Copied! copy

The starting position rhymes with "fun, door".

setKnightPosition (y, x) Copied! copy

Move the WK1 to new starting coordinates.

getLegalMoves(y, x) Copied! copy

Punch in a pair of coordinates, return an array of legal destinations for the next move.

knightsTravails(y, x) Copied! copy

Give coordinates from [0,0] to [7,7] as [y, x]. App will calculate the minimum number of moves to arrive there.
